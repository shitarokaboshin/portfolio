# Portfolio (Vite + React)

This is a minimal React portfolio scaffold that mirrors the structure shown in the screenshots: hero, projects, trips (gallery), and certifications.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Files of interest

- [index.html](index.html)
- [package.json](package.json)
- [src/main.jsx](src/main.jsx)
- [src/App.jsx](src/App.jsx)
- [src/components/Hero.jsx](src/components/Hero.jsx)
- [src/components/Projects.jsx](src/components/Projects.jsx)
- [src/components/Trips.jsx](src/components/Trips.jsx)
- [src/components/Certifications.jsx](src/components/Certifications.jsx)

You can replace the placeholder images in `src/data.js` with your own images and update the text to match your content.
